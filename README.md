# LookLook 

此app用来看知乎日报，网易新闻和每日看看，很好的展示了MVP架构，使用了RxAndroid和retrofit，逻辑清晰，是很好的练手小项目。我在原有的基础上添加了大量的注释，删除了一些冗余的代码和文件，修改了很多Bug，但是里面还有一些工具类和View还没有了解其具体功能，以后可能会继续添加
